The main idea:

  • The application provides courses reservation service for the users.
  • Users need to register then choose the courses suite them.
  • There are registration and login forms, with all necessary validations.
  • The database is updated dynamically for users and courses.

1- Registration form:

  • User submits required information of username, email and password

2 Login form:

  • Email and password are required to sign-in

2- Users:

  • Users can easily choose and reserve the suitable courses in the member area page.

3- Admin:

  • Admin checks out courses, users and their reservation.
